#1.2 History of Python

History and Evolution of Python


Python was created by Guido van Rossum in the late 1980s and early 1990s at the National Research Institute for Mathematics and Computer Science in the Netherlands. Influenced by languages like ABC, Modula-3, C, C++, Algol-68, SmallTalk, Unix Shell, and various scripting languages, Python has continuously evolved over the decades.

______________________________________________________________________________________

Key Milestones:

  • 1991: Python 1.0 was released, introducing several functional programming tools.

  • 2000: Python 2.0 was launched by the BeOpen Python Labs team, introducing list comprehensions and other improvements.

  • 2010: Python 2.7, the last major update of Python 2.x, remained widely used but was officially supported until 2020.

  • Present: Python 3.x, including versions like Python 3.6.4, introduced enhanced features, such as improved string representation and more efficient memory handling.

______________________________________________________________________________________

Licensing and Community Support:

Though copyrighted, Python's source code is freely available under the GNU General Public License (GPL), similar to Perl. Today, a dedicated core development team maintains Python, continuing the vision initially led by Guido van Rossum.


Pythons Growing Influence:

From data science to web development, Python has solidified itself as a versatile and powerful programming language. Surprisingly, Python predates other popular languages like Java, R, and JavaScript, yet remains one of the most widely adopted languages today.
